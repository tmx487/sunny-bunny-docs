[Класс](#https://learn.microsoft.com/en-us/dotnet/csharp/language-reference/keywords/class) - "чертеж" будущего экземпляра. Является ссылочным типом. Может содержать следующие члены:
- Constructors
- Constants
- Fields
- Finalizers
- Methods
- Properties
- Indexers
- Operators
- Events
- Delegates
- Classes
- Interfaces
- Structure types
- Enumeration types
